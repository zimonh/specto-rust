How it works

What is a spectrogram?

A spectrogram visualises the frequency content of audio over time. The horizontal axis is time, the vertical axis is frequency, and brightness shows how loud each frequency is at each moment. It is computed using the Fast Fourier Transform (FFT).

Circle of Fifths colour scheme

Instead of brightness, this mode colours each frequency by its musical pitch class (C, C#, D … B) and maps it around the Circle of Fifths — a layout where harmonically related keys sit close together. The result is that chords and harmonics share similar hues, making musical structure visible at a glance.

Multi-band resolution

Low frequencies need a larger FFT window to resolve individual notes, while high frequencies can use a smaller one for better time resolution. This viewer uses a technique inspired by Constant-Q analysis — different frequency bands can be rendered with different FFT sizes simultaneously, giving the best of both worlds.

Built with

Rust + WebAssembly for parallel FFT computation, with source on GitHub.

💡 Gain slider — adjusts brightness. Turn it up if the spectrogram looks dark, down if it's washed out.

🎨 Hue rotate — shifts all note colours together while keeping their relationships accurate. The Circle of Fifths mapping is relative, so rotating never breaks harmonic accuracy — C and G will always be close in hue no matter where you rotate to.
